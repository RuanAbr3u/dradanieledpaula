# 🩺 Landing Page — Dra. Daniele D'Paula

Landing page institucional desenvolvida para a **Dra. Daniele D'Paula**, fisioterapeuta especialista em **Fisioterapia** e **RPG (Reeducação Postural Global)**, com foco em uma experiência moderna, elegante e responsiva.

O projeto foi desenvolvido utilizando **React**, **TypeScript**, **Vite** e **Tailwind CSS v4**, priorizando performance, acessibilidade e uma excelente experiência do usuário.

## 💻 Instalação

Clone o projeto:

```bash
git clone https://github.com/SEU-USUARIO/dra-daniele-landing.git
```

Entre na pasta:

```bash
cd dra-daniele-landing
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

---

## 📦 Build

```bash
npm run build
```

Preview da versão de produção:

```bash
npm run preview
```
